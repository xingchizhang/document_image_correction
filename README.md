# document_image_correction
文档图像矫正
# 项目描述
对发生透视变换的文档图像进行几何校正处理，得到规范的文档图像。  
几何校正的目的是把发生了透视变换的目标变换为具有真实比例和角度的目标，如下图所示：  
左图中的地板砖经过透视变换之后，不再是正方形，经过校正之后，得到右侧图像，其中的地板砖被校正为正方形：
